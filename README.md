# Cough Sound Classification using Machine Learning

## Objective
To classify cough sounds as healthy or unhealthy using audio processing.

## Technologies Used
- Python
- Librosa
- Scikit-learn

## Methodology
- Extract MFCC features from cough audio
- Train Random Forest model
- Predict cough category

## Result
Model successfully predicts cough type based on audio input.

## Application
Useful for early detection of respiratory diseases.
